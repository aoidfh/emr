🏥 EMR 병원차트 프로그램

실제 병원 업무 흐름을 기반으로 구축된 웹 기반 EMR(전자의무기록) 시스템입니다.
접수실, 진료실, 수납실 등 병원 내 부서별 기능을 UI로 구현하고, MySQL 연동을 통해 환자 정보와 진료 데이터를 처리합니다.

📌 주요 기능

접수실:

환자 검색, 접수 처리, 접수 리스트 출력

과거 진료 이력 조회

진료실:

진료 기록 작성, 진료처방 UI 구성

수납실:

진료비 산출 기능 구현 (결제 기능은 제외)

팝업창 기능:

약 20여 개 UI 구성

환자조회, 접수정보, 처방코드 검색 등 일부 팝업은 MySQL과 연동

나머지 팝업은 이미지 기반 UI로 구성

병원관리 / 기초정보 관리:

일부 항목에 대해 DB 연동 및 기본 CRUD 구현

로그인 / 메인메뉴:

커스텀 로그인 UI 구성 (실제 인증 미연동)

메인 메뉴 클릭 시 페이지 전환만 가능

MySQL 연동:

환자 이름 검색, 접수 처리, 진료 내역 등 실시간 DB 저장/조회 처리

🛠 사용 기술
영역	기술 스택
Frontend	HTML, CSS, JavaScript
Backend	없음 (Express 미사용, DB 직접 연동)
Database	MySQL
기타	Live Server (로컬 테스트용)
⚠ 참고 사항

현재는 로컬 환경 미리보기를 위한 UI 위주 구성이며, 서버 연동은 포함되어 있지 않습니다.

실제 병원 환경 및 보안 이슈로 인해 일부 기능과 경로는 축약·가공되어 업로드되었습니다.

본 프로젝트는 병원 납품용 시스템으로 개발되었으며, 박준영은 접수/진료/수납 업무 UI 구현과 MySQL 연동을 포함한 초기 개발을 단독 수행하였습니다.

이후 회사 사정으로 퇴사하면서 후속 개발에는 참여하지 못하였습니다.

전체 프로젝트 구조와 DB 연동 상세 로직은 요청 시 별도로 제공 가능합니다.
