# 🏥 EMR 병원차트 프로그램

> 실제 병원 업무 흐름을 반영한 웹 기반 EMR(전자의무기록) 시스템입니다.  
> 접수실, 진료실, 수납실 등 병원 내 각 부서의 업무를 웹 UI로 구현하고, MySQL 연동을 통해 실시간 환자 정보를 처리합니다.

---

## 📌 주요 기능

- 접수실 UI: 환자 검색, 접수 처리
- 진료실 UI: 진료기록 작성, 과거 이력 조회
- 수납실 UI: 진료비 계산 및 결제 처리
- 팝업창 UI: 세부 내역 조회, 기타 부가 기능
- MySQL 연동을 통한 실시간 DB 저장 및 검색

---

## 🛠 사용 기술

| 영역 | 기술 |
|------|------|
| Frontend | HTML, CSS, EJS |
| Backend | Node.js (Express는 사용하지 않음) |
| DB | MySQL |
| 기타 | JavaScript, Live Server 환경 확인용

---

## ⚠ 참고 사항

- 현재는 로컬 미리보기용으로 구성되어 있으며, 서버 연동 환경은 포함되어 있지 않습니다.
- 파일 경로 및 일부 기능은 실제 병원 환경과 보안 이슈로 인해 축약·가공되어 업로드되었습니다.
- 전체 프로젝트 구조와 DB 연동 로직은 요청 시 별도로 제공 가능합니다.
